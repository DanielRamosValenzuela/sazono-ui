# Sazono UI

Frontend principal de Sazono construido con Next.js.

## Enfoque

- App Router usando `src/app`
- estructura FSD pragmatica bajo `src/`
- separacion temprana entre experiencia `qr` y `staff`

## Estructura inicial

```text

  app/
  pages/
  widgets/
  features/
  entities/
  shared/
```
